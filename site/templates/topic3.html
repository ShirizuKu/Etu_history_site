История СПбГЭТУ «ЛЭТИ» в современный период (1991–настоящее время)

1. Трансформация в условиях новых реалий

  • В 1992 году институт получил статус университета и новое название — Санкт-Петербургский государственный электротехнический университет «ЛЭТИ».
  • Созданы инновационные структуры:
    • Технопарк «ЛЭТИ» (2010) для поддержки стартапов в области IT и робототехники.
    • Центр нанотехнологий с «чистыми комнатами» для разработки микроэлектронных устройств (2015).

2. Научные прорывы XXI века

Ключевые направления:

  • Искусственный интеллект и Big Data:
    • Разработка нейросетевых алгоритмов для медицинской диагностики (проект с НИИ онкологии им. Петрова).
    • Системы анализа данных для «умных городов» (совместно с ПАО «Ростелеком»).
  • Биомедицинская инженерия:
    • Создание биосенсоров для раннего выявления COVID-19 (2020).
    • Имплантируемые нейростимуляторы (коллаборация с НМИЦ им. Алмазова).
  • Квантовые технологии:
    • Эксперименты с квантовой криптографией в партнёрстве с РКЦ (2022).

3. Международная интеграция

  • Участие в программах:
    • Erasmus+ (обмен с TU Munich, KTH Stockholm).
    • BRICS Network University (совместные исследования в области энергетики).
  • Открытие англоязычных магистерских программ:
    • «Internet of Things Engineering».
    • «Biomedical Systems and Technologies».

4. Образовательные инновации

  • Цифровизация учебного процесса:
    • Запуск платформы «ЛЭТИ-Онлайн» с VR-лабораториями (2021).
    • Программы Micro-Degree по кибербезопасности и AI.
  • Новые факультеты:
    • Факультет искусственного интеллекта и промышленного IoT (2023).
    • Кафедра устойчивой энергетики (2022).

5. Культура и социальные проекты

  • Студенческий кампус:
    • Открытие коворкинг-центра «Точка кипения» (2018).
    • Эко-инициативы: солнечные панели на крышах корпусов (2023).
  • Спортивные достижения:
    • Команда по робототехнике — призёр Eurobot-2023.

6. Стратегия развития до 2030

  • Вхождение в топ-200 мировых рейтингов QS и THE.
  • Создание «цифрового двойника» университета для управления ресурсами.
  • Проект «ЛЭТИ-Арктика»: разработка автономных энергосистем для Крайнего Севера.
